ubuntu下忘记mysql5.7账号密码

cd /etc/mysql sudo cat debian.cnf 通过里面的user password即可进入mysql,然后再通过该账号修改密码或创建新账号

如何从github上下载单个文件

首先获取raw地址path：然后 curl -O path 或者 wget path 例如： curl -O https://raw.githubusercontent.com/python/cpython/master/Objects/tupleobject.c wget https://raw.githubusercontent.com/python/cpython/master/Objects/listobject.c

ubuntu在终端打开文件夹

nautilus, 直接打开Home nautilus . 打开当前文件夹 nautilus [path] 打开path路径文件夹
